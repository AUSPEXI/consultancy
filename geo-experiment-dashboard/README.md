# GEO Lab — Cinematic Video Recorder

A Vite + React + TypeScript studio for turning a GEO Lab experiment into a
finished YouTube video. It loads a **storyboard project** (`*.project.json`,
generated by `geo-lab/scripts/build-storyboard.mjs`), plays the programme —
intro → storyboards (with narration + b-roll cutaways) → outro — and
**screen-records it to a downloadable `.webm`** ready for upload.

This app is provider-agnostic: it records whatever you load into it. Voiceovers
are produced in ElevenLabs and b-rolls in Google Flow, then uploaded at runtime
(stored locally in IndexedDB, never committed).

## Run locally

**Prerequisites:** Node.js 18+

```bash
npm install
npm run dev          # http://localhost:3000
```

`npm run lint` typechecks (`tsc --noEmit`); `npm run build` produces a production
bundle.

## Workflow

1. **Load Project** — pick the experiment's `storyboard.project.json` (or use the
   built-in experiment-001 template).
2. Upload the **voiceover** (ElevenLabs) and the **b-roll** clips (Flow) for the
   panels that call for them; optionally an intro/outro.
3. **Sync** the panels to the voiceover by editing each panel's start/end time
   (type the seconds, or snap to the playhead).
4. **Capture Take** — the stage goes fullscreen, you pick the window + share
   audio, and it plays the whole programme through and auto-downloads the recording.
5. Convert `.webm` → `.mp4` if you prefer:
   `ffmpeg -i take.webm -c:v libx264 -crf 18 -pix_fmt yuv420p -c:a aac -b:a 192k take.mp4`

See `STORYBOARD_SCHEMA.md` for the project file format.
