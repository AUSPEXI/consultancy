Rewriting the same facts as direct definitions raised AI citation rate from 28.1% to 96.9% across four engines — a +68.8pp jump from a single structural change. In this experiment I tested whether definitional structure ("X is a Y that does Z") gets cited more than hedged narrative prose for "what is X" queries. Same brand (NovaCRM), same facts, same ~175-word length — the only variable was sentence framing.
