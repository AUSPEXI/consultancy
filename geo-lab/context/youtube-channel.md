# YouTube Channel Strategy

The channel turns GEO experiments into trustworthy, reproducible videos. The
through-line: **"I tested it so you don't have to — here's the data."**

## Why this works
Nobody else is running controlled, statistical A/B tests on LLM citation and
showing the raw numbers. That is a defensible content moat and it directly
demonstrates the expertise Auspexi sells.

## Video format (8–14 min)
1. **Hook (0:00–0:20)** — the question + a teaser of the surprising result.
   "Does adding one statistic make AI 3× more likely to cite you? I ran 256 probes."
2. **The hypothesis (0:20–1:00)** — what we're testing and why it might matter.
3. **The method (1:00–3:00)** — variants on screen, the probe, n, the four engines.
   Show the pre-registered DESIGN.md. This is the trust-builder.
4. **The run (3:00–5:00)** — watch probes execute, raw responses on screen.
5. **The result (5:00–8:00)** — rates, the chart, the p-value, plain conclusion.
6. **Threats to validity (8:00–9:30)** — what could be wrong. Builds authority.
7. **What it means for you (9:30–end)** — the practical takeaway + soft Auspexi CTA.

## Series ideas (each = a playlist)
- **"GEO A/B"** — the core weekly experiment series.
- **"Engine wars"** — same content, how differently do GPT/Gemini/Claude/PPLX cite it?
- **"Myth-busting AI SEO"** — test popular advice; show what's bunk.
- **"Live index diaries"** — slow-mode experiments tracked over weeks.

## Per-video deliverables (the video-producer agent outputs these)
- `script.md` — spoken script with on-screen-text cues and b-roll notes.
- `titles.md` — 5 title options, ranked, each ≤ 60 chars, curiosity + number.
- `thumbnail.md` — concept brief (big number, two variants, an arrow).
- `description.md` — SEO description, chapter timestamps, links, ™ on first mention.
- `pinned-comment.md` — the raw-data link + reproduction steps (reinforces trust).

## Title patterns that fit the brand
- "Does [lever] actually get you cited by AI? (n=[N] test)"
- "I ran [N] AI citation probes. [Surprising result]."
- "[Engine] cites this. [Engine] ignores it. Here's why."

## Cadence
One fast-mode experiment per week → one core video. One flagship live-mode
video per quarter. Re-cut shorts from the result charts.

## Hard rules
- The thumbnail number must match the video's real number. No clickbait deltas.
- Always link the experiment folder (or a public mirror) so viewers can reproduce.
- If the result is null, the title says so honestly — "Turns out it doesn't matter."
